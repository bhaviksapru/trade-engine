Trade
Engine

NinjaTrader charges approximately $100 per month for live trade execution — a fee that exists purely for routing orders to your broker through their integration layer. This project replaces that entirely.

NinjaTrader is kept as a signal generator only, which is free. Every buy/sell signal it fires goes to AWS via HTTPS, where a Step Functions state machine owns the full trade lifecycle: risk checks, order placement at Interactive Brokers, fill monitoring, stop management, and position closing. A live dashboard lets you watch it all in real time.
